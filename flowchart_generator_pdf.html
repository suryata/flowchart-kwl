Keterangan

Frontend (Next.js) — Halaman course di /courses/[id], modal form, validasi input, download file
API Endpoint — POST /api/course/generate-kwl-pdf dengan autentikasi JWT
Backend (Django) — Validasi data, generate PDF dengan ReportLab, return bytes
Mode Default — 3 pertanyaan standar KWL otomatis berdasarkan topik
Validasi — Topik maks 80 karakter, pertanyaan maks 3 per section, batas karakter per pertanyaan dinamis
