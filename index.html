Keterangan

Frontend (Next.js) — Tombol Generate KWL di setiap kartu topik, modal form, validasi input, download file
API Endpoints — GET /api/course/kwl-form-questions/:topicId untuk fetch pertanyaan tersimpan, POST /api/course/generate-kwl-pdf untuk simpan & generate
Backend (Django) — Validasi data, simpan ke tabel KWLFormQuestion, generate PDF dengan ReportLab
Penyimpanan — Pertanyaan disimpan di tabel terpisah (KWLFormQuestion), tidak terkait fitur essay existing
Mode Default — 3 pertanyaan standar KWL otomatis berdasarkan nama topik
Validasi — Topik maks 80 karakter, pertanyaan maks 3 per section, batas karakter dinamis per pertanyaan
